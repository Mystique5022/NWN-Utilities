What this can do Nvidia Cards only. Might not work for all cards. (Might take a few tries)

DISCLAIMER: If you damage your system doing this, I am not responsible, but I have a hard time seeing how you could really do that.

  1. Obtain your current desktop resolution. Right click on Desktop -> Screen Resolution -> Note the value. (i.e. 1024x768)
  2. Launch file explorer, and navigate to NWN Root directory where nwmain.exe is ( Might be close to C:\Program Files (x86)\Neverwinter Nights\NWN. )
  3. Open nwn.ini using notepad. Find the line 'AllowWindowedMode=' This parameter should be set to 1. Now find and make sure the 'FullScreen=' parameter is equal to 0.

    4. NWN.ini -> The values in this example are relative and should be used as a pattern reference only.

  4. In that file, find the parameter 'Height=' Try something like 700 (**RELATIVE**, remember to subtract from YOUR Resolution)
  5. Same file, under it, set the 'Width=' parameter equal to something smaller than 1024 (**RELATIVE**, remember to subtract from YOUR Resolution). Try something like 1018.
  6. Save this file, and close it. But leave the folder open anyways, since you might need to revisit this.

    7. Make the matching custom NVidia Resolution (* NOTE - If something doesn't work, it might not be compatible with your video card / monitor * )

  7. Open NVIDIA control panel from the destop. Right click on your wallpaper -> NVIDIA Control Panel
  8. Go to Change resolution under the Display menu.
  9. Under resolution selection box, make sure the value you noted in step 1 is what is already selected. It should also say (native) next to it. Find and click the Customize button under it..
  10. There might be a warning, click yes, if so. "Enable resolutions not exposed by the display" should be checked.
  11. Click Create Custom Resolution. Enter the value you computed in step 4 as 'Horizontal Pixels' and the value in step 5 as 'Vertical lines' in their respective boxes in this dialog menu.
  12. Click Test. It will try to use that resolution. Wait, and don't touch your mouse. If your monitor black screens, nothing in this guide will not work for you, but don't worry, it should reset in 15 seconds automatically.
  13. If it succeeds, click OK to continue adding the resolution. Now, make sure the ORIGINAL resolution is selected. i.e. 1024x768 (native)
  14. Launch NWN. Note results. Repeat all of these steps to tweak and perfect to your OCD pleasures.
